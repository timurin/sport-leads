# ADR-011: Media номенклатуры

Статус: принят в v0.8.7.

## Решение

Изображение номенклатуры хранится как отдельная `NomenclatureMedia`: metadata находится в PostgreSQL, бинарный файл — в локальном ignored storage по непрозрачному UUID storage key. Разрешены JPEG, PNG, WebP и SVG размером до 10 MB. В карточке может быть одно главное изображение; порядок задаётся `sort_order`.

API не принимает произвольный путь и проверяет принадлежность storage key безопасному корню перед чтением или удалением. Удаление удаляет metadata и файл, а удаление номенклатуры каскадирует metadata; production object storage, CDN, image transforms и права доступа остаются будущими решениями.

## Границы

Контур не включает модели изделий, лекала, размерные сетки, спецификации, производство, склад, цены, штрихкоды или интеграцию с 1С.
