# Release checklist

## Техническая проверка

- [ ] backend tests;
- [ ] frontend unit tests;
- [ ] TypeScript;
- [ ] lint;
- [ ] production build;
- [ ] `python scripts/check_project.py`;
- [ ] OpenAPI и Alembic, если затронуты;
- [ ] `git diff --check`.

## Ручная проверка

- [ ] ключевой пользовательский сценарий;
- [ ] визуальная проверка изменённых страниц;
- [ ] локальное открытие HTML-статусов;
- [ ] отсутствие внешних зависимостей в автономных HTML;
- [ ] UTF-8 и внутренние ссылки.

## Документация и Git

- [ ] обновлены затронутые канонические документы;
- [ ] roadmap/ERP статусы подтверждены кодом и тестами;
- [ ] commit, tag и push выполнены только после разрешения владельца проекта;
- [ ] release/checkpoint документ создан при необходимости.
