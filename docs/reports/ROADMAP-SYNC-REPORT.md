## Расхождения GitHub и локальной копии

`origin/main` указывает на merge checkpoint `v0.5.0`, а текущий `HEAD` локальной feature-ветки опережает его на четыре коммита. Рабочая копия дополнительно содержит незакоммиченные изменения списка лидов, dev-seed, тестов и документации.

До начала этой задачи рабочее дерево уже было dirty и содержало изменения бизнес-кода и untracked-файлы. Они сохранены без удаления или перезаписи. Поэтому общий `git diff` рабочей копии не состоит только из документации, хотя изменения, внесённые именно задачей синхронизации, ограничены перечисленными документами.
