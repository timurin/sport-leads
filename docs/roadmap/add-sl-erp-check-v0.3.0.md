## Проверки
Выполнить:

- `python scripts/check_project.py`
- `docker compose config --quiet`
- `git diff --check`

Если доступны:

- `cd frontend`
- `npm run lint`
- `npm run build`

Дополнительно проверить:

- файл структуры существует;
- код документа — `SL-ERP-CHECK-v0.3.0`;
- ссылка на roadmap работает;
- ссылки из `docs/README.md` работают;
- старый код `SL-ERP-CHECK-v1` сохранён как исторический;
- `git diff` содержит только документацию.
