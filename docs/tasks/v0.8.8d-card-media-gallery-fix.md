# v0.8.8d — Media gallery fix

## Цель

Исправить загрузку и отображение изображений в карточке номенклатуры.

## Проблема

Сейчас:

- загрузка не работает;
- изображения не появляются;
- отсутствуют миниатюры;
- media-блок не соответствует HTML-эталону.

## Сделать

Подключить существующий API NomenclatureMedia и реализовать:

- выбор файла;
- загрузку;
- состояния загрузки и ошибки;
- отображение миниатюры сразу после успешной загрузки;
- загрузку существующих изображений при открытии карточки;
- главное изображение;
- назначение другого изображения главным;
- сортировку;
- удаление;
- alt-текст;
- пустое состояние;
- placeholder;
- проверку JPG, PNG и WEBP;
- лимит размера согласно backend;
- корректный URL выдачи изображения.

## UI

Повторить media-блок из HTML-эталона:

- сетка миниатюр;
- отметка «Главное»;
- отдельная плитка «Добавить»;
- кнопка «Загрузить»;
- drag-and-drop, если он уже предусмотрен;
- адаптивная сетка;
- миниатюра главного изображения в шапке карточки, если это поддерживается текущей структурой.

Не использовать внешние изображения из HTML-эталона. Эталонные изображения служат только примером визуального расположения.

## Проверки

- загрузка изображения;
- повторное открытие карточки;
- отображение миниатюр;
- назначение главного;
- сортировка;
- удаление;
- несуществующий файл;
- слишком большой файл;
- неподдерживаемый MIME;
- frontend/backend tests;
- TypeScript;
- ESLint;
- build;
- git diff --check.
