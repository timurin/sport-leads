## Purpose

Глобальный журнал операций фиксирует **участие модели** в продажах и производстве.

| Event | Write to journal? |
|---|---|
| Model selected on a sales order / sold | **Yes** |
| Model never used in a sale / production | **No** |
| Production / technical card uses the model | **Yes** (when production contour ready) |
