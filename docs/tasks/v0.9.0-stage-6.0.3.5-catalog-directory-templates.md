# Task — Catalog directory template etalons (`6.0.3.5`)

**Roadmap:** `6.0.3.5` (annotates `6.1.7` / `6.1.8`)  
**Iteration:** `v0.9.0` / `2026-07-22`

## Decision

Product-model routes are the **primary templates** for settings directories, sections, and categories:

| Route | Contract | Role |
|---|---|---|
| `/settings/catalogs/product-models` | `DS-PT-02-CATALOG` | Catalog **list** etalon |
| `/settings/catalogs/product-models/[id]` | `DS-PT-08-CATALOG` | Catalog **card** etalon (versioned) |

Simple non-versioned cards remain **PT-05**. CRM lists remain **PT-02** (`/sales/clients`).
